除了正北（True North）和磁北（Magnetic North），还有一个叫做**网格北（Grid North）**。

### 1. **正北（True North）**
   - 指向地球自转轴的北极点，也称为地理北极。
   - 是地图和导航中最常用的参考方向。

### 2. **磁北（Magnetic North）**
   - 指向地球磁场的北极点（磁北极）。
   - 磁北极的位置会随着时间变化（地磁偏角）。

### 3. **网格北（Grid North）**
   - 是地图上的垂直网格线所指向的北方。
   - 通常用于平面坐标系统（如 UTM 坐标系）。
   - 网格北和正北之间可能存在一定的偏差，称为**网格偏角**。

### 总结
这三种“北”在导航和地图学中都有不同的用途，具体使用哪种取决于场景需求。例如：
- **正北**：用于天文导航和全球定位。
- **磁北**：用于传统的磁罗盘导航。
- **网格北**：用于地图测绘和坐标系统。

在地底下导航时，由于 GPS 信号无法穿透地面，传统的基于卫星的导航方法（如正北）不可用，而磁场可能受到地质结构的干扰，因此需要根据具体场景选择合适的参考方向：

### 1. **磁北（Magnetic North）**
   - 如果地底的磁场干扰较小，可以使用磁北作为参考方向。
   - 使用磁罗盘或磁力计（如手机内置的磁力传感器）来确定方向。
   - 需要注意地磁异常区域（如矿区或含铁矿石的地层），可能导致磁北方向不准确。

### 2. **惯性导航（基于陀螺仪）**
   - 在地底下，惯性导航系统（INS）是更可靠的选择。
   - 使用陀螺仪和加速度计来跟踪设备的运动和方向。
   - 惯性导航不依赖外部信号，但会随着时间累积误差（漂移）。

### 3. **网格北（Grid North）**
   - 如果你在地底下使用的是预先绘制的地图（如矿井或隧道的平面图），可以使用网格北作为参考方向。
   - 地图上的网格线提供了一个固定的方向基准。

### 4. **自定义参考方向**
   - 在某些场景下，可以人为定义一个参考方向（如矿井入口的方向作为“北”）。
   - 这种方法适用于封闭环境，且不依赖地磁或地理方向。

### 推荐方法
- **矿井或隧道导航**：结合惯性导航和预先绘制的地图（网格北）。
- **地质勘探**：使用磁北，但需校正地磁偏角和考虑地磁干扰。
- **复杂环境**：使用惯性导航与其他传感器（如超宽带 UWB 或 RFID）结合，进行多传感器融合定位。
